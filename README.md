

🔐 SecurePass

Android Password Strength Checker Application

📌 Project Overview

SecurePass is an Android application that evaluates the strength of a password based on various security criteria. It helps users create stronger and more secure passwords by providing real-time feedback and a strength rating (Weak, Medium, Strong).

This project was developed as part of the Android Application Development course.

🎯 Objectives
	•	Analyze password strength based on:
	•	Length
	•	Uppercase letters
	•	Lowercase letters
	•	Numbers
	•	Special characters
	•	Detect common or predictable patterns
	•	Provide a visual strength indicator
	•	Encourage users to create secure credentials



⚙️ Features

✔ Real-time password analysis
✔ Strength score display (Weak / Medium / Strong)
✔ Clean and user-friendly interface
✔ Instant feedback while typing



🛠 Technologies Used
	•	Kotlin
	•	Android Studio
	•	XML (UI Design)
	•	Git & GitHub



📱 How It Works
	1.	User enters a password.
	2.	The app analyzes:
	•	Character diversity
	•	Password length
	•	Presence of numbers and symbols
	3.	A strength rating is displayed.
	4.	Suggestions may be shown to improve weak passwords.
